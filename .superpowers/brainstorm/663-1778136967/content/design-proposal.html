二、数据模型设计（AI 知识库友好）

📁 categories          ← 分类/手册（如"房源管理"）
  ├── id, title, sort, parent_id
  ├── icon, description
  │
📄 articles            ← 文章/章节（如"新增房源"）
  ├── id, category_id, title, content
  ├── content_html, content_json (富文本)
  ├── sort, status, version
  │
🏷️ tags                ← 标签（便于 AI 检索）
  ├── id, name
  │
📎 article_tags        ← 文章-标签关联
  ├── article_id, tag_id

💡 content_json 同时存纯文本版本 → AI 知识库直接消费
